🎯 이 장에서 배우는 것

학습 목표

📌 핵심 개념

개념설명
folium파이썬으로 인터랙티브 지도를 만드는 라이브러리. 마우스로 확대·축소·이동 가능
위도·경도지구 위의 모든 위치를 나타내는 숫자. 서울: 위도 37.5, 경도 127.0
지오코딩주소 → 위도·경도로 변환하는 것. "경기도 고양시 덕양구 ..." → (37.6, 126.8)
MarkerCluster가까운 마커들을 묶어서 숫자로 표시. 줌인하면 개별 마커로 풀림
💡 오늘 쓸 라이브러리

3
여러 마커 찍기 + 군집화

639개 사적지를 for문으로 한번에 찍고, MarkerCluster로 깔끔하게 묶어봐요.

Python
import pandas as pd
import glob
from folium.plugins import MarkerCluster

# 데이터 불러오기
files = glob.glob('/content/drive/MyDrive/**/독립기념관_국외사적지*.csv', recursive=True)
df = pd.read_csv(files[0], encoding='cp949')

# 지도 + 클러스터 생성
m = folium.Map(location=[30, 100], zoom_start=3)
cluster = MarkerCluster().add_to(m)

# for문으로 모든 행에 마커 찍기
# .add_to(cluster) — m 대신 cluster에 추가하면 군집화!
for idx, row in df.iterrows():
    folium.Marker(
        location=[row['위도'], row['경도']],
        popup=row['사적지명(국문)'],
        tooltip=row['사적지명(국문)']
    ).add_to(cluster)

display(m)
실행 결과
숫자가 적힌 원으로 마커들이 묶여요. 중국 쪽에 큰 원(373), 미국 쪽에 작은 원(101)이 보여요. 줌인하면 원이 점점 풀리면서 개별 마커가 나타나요!
💡 중국 쪽을 줌인해봐요. 상하이 임시정부, 윤봉길 의거 장소 등 구체적인 사적지들이 보여요!

5
지오코딩 코드 실행하기

아래 코드에서 REST API 키만 입력하고 실행하면 돼요.
약 2분 후 위도·경도가 추가된 CSV가 드라이브에 저장돼요.

Python
import requests

KAKAO_API_KEY = '여기에_REST_API_키_입력'   # ← 키 입력!

def kakao_geocoding(address):
    url = 'https://dapi.kakao.com/v2/local/search/address.json'
    headers = {'Authorization': f'KakaoAK {KAKAO_API_KEY}'}
    result = requests.get(url, headers=headers, params={'query': address}).json()
    if result['documents']:
        return float(result['documents'][0]['y']), float(result['documents'][0]['x'])
    return None, None

# 동물병원 데이터 불러오기
files2 = glob.glob('/content/drive/MyDrive/**/경기도_고양시_동물병원현황.csv', recursive=True)
df2 = pd.read_csv(files2[0], encoding='cp949')
df2['동명'] = df2['사업장소재지(도로명)'].str.extract(r'\(([^)]+)\)')

# 위도·경도 추가 (약 2분 소요)
df2['위도'] = df2['사업장소재지(도로명)'].apply(lambda x: kakao_geocoding(x)[0])
df2['경도'] = df2['사업장소재지(도로명)'].apply(lambda x: kakao_geocoding(x)[1])

print(f'변환 완료: {df2["위도"].notna().sum()}/{len(df2)}개')

# 드라이브에 저장 — 다음에 다시 지오코딩 안 해도 돼요!
df2.to_csv('/content/drive/MyDrive/동물병원_위도경도.csv', index=False, encoding='utf-8-sig')
print('저장 완료!')
