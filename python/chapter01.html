📚 핵심 개념 — 구글 코랩

개념 구글 코랩(Google Colab)이란?

구글이 무료로 제공하는 파이썬 실습 환경입니다. 설치 없이 인터넷 브라우저에서 바로 파이썬 코드를 실행할 수 있습니다.

특징설명
🌐 브라우저 기반설치 없이 인터넷만 있으면 바로 사용 가능
☁️ 구글 드라이브 연동작업 내용이 자동으로 저장됨
📦 패키지 미리 설치pandas, matplotlib 등 주요 패키지 기본 탑재
🆓 무료구글 계정만 있으면 무료로 사용 가능
접속 방법: 구글 검색에서 "구글 코랩" 검색 → colab.research.google.com 접속 → 구글 계정 로그인
개념 셀(Cell)이란?

코랩은 셀 단위로 코드를 작성하고 실행합니다.

셀 종류용도실행
코드 셀파이썬 코드 작성 및 실행▶ 버튼 클릭 또는 Shift+Enter
텍스트 셀설명, 제목 등 문서 작성더블클릭으로 편집
중요: 코랩은 런타임이 연결 끊기면 작업이 초기화됩니다. 중요한 파일은 구글 드라이브에 저장해두세요.

🖨️ 핵심 개념 — print() 함수

개념 print() — 화면에 출력하기

print()는 괄호 안의 내용을 화면에 출력하는 함수입니다.

핵심 규칙: 따옴표가 있으면 그대로 출력, 따옴표가 없으면 변수 값 출력 
print('출력할 내용')   # 따옴표 있음 → 그대로 출력
print(100)            # 숫자는 따옴표 없이
print(3.14)           # 소수점도 가능
실행 결과
출력할 내용 100 3.14

🔨 따라하기

1
여러 값 한 번에 출력하기

쉼표(,)로 여러 값을 한 줄에 출력할 수 있습니다. 자동으로 사이에 공백이 추가됩니다.

print('이름:', '홍길동')
print('나이:', 17)
print('좋아하는 가수:', 'BTS')
실행 결과
이름: 홍길동 나이: 17 좋아하는 가수: BTS

⚠️ 주의할 점

따옴표를 빠뜨리면 오류!
문자열은 반드시 따옴표로 감싸야 합니다. print(안녕)이라고 쓰면 NameError가 납니다.
대소문자 구분!
Print(), PRINT()는 오류가 납니다. 반드시 print()로 소문자로 써야 합니다.

📦 핵심 개념 — 변수

개념 1 변수(Variable) — 데이터를 담는 상자

변수는 데이터를 저장해두는 이름이 붙은 상자입니다.
= 기호로 값을 저장합니다. (수학의 등호와 다릅니다!)

# 변수 만들기 (변수명 = 값)
name = '홍길동'      # 문자열 저장
age = 17            # 정수 저장
height = 172.5     # 소수 저장
is_student = True  # 참/거짓 저장

print('이름:', name)     # 따옴표 없음 → 변수 값
print('나이:', age)
print('키:', height)
실행 결과
이름: 홍길동 나이: 17 키: 172.5
개념 2 변수 이름 짓기 규칙
규칙좋은 예나쁜 예
영어 소문자 권장name, ageName, AGE
띄어쓰기 불가 → _ 사용my_namemy name
숫자로 시작 불가score11score
의미 있는 이름 사용student_namea, x1

🔨 따라하기

1
변수로 계산하기

변수에 저장된 값으로 계산을 할 수 있습니다.

korean = 90
math = 85
english = 92

total = korean + math + english
average = total / 3

print('총점:', total)
print('평균:', average)
실행 결과
총점: 267 평균: 89.0
2
변수 값 바꾸기

같은 변수에 새로운 값을 넣으면 이전 값은 사라집니다.

score = 70
print('처음 점수:', score)

score = 85       # 값 변경
print('바뀐 점수:', score)

score = score + 10  # 기존 값에 10 더하기
print('최종 점수:', score)
실행 결과
처음 점수: 70 바뀐 점수: 85 최종 점수: 95

⚠️ 주의할 점 — = vs ==

=  대입 연산자
오른쪽 값을 왼쪽 변수에 저장합니다.
x = 5
name = '홍길동'
→ x라는 상자에 5를 넣는다
==  비교 연산자
왼쪽과 오른쪽이 같은지 비교합니다. 결과는 True/False.
x == 5    # True
x == 3    # False
→ x 상자 안이 5인지 물어본다
# 예시로 비교해봐요
score = 90           # = : score에 90을 저장

print(score == 90)   # == : 90이 맞냐? → True
print(score == 100)  # == : 100이 맞냐? → False
❌ 자주 하는 실수
if score = 90:  ← 오류! 조건을 비교할 때 =을 쓰면 안 돼요.
if score == 90:  ← 정상! 비교할 때는 반드시 ==을 써요.

📋 핵심 개념 — 리스트

개념 리스트(List) — 여러 값을 순서대로 저장

리스트는 여러 개의 값을 하나의 변수에 순서대로 저장하는 방법입니다.
대괄호 [ ]로 만들고, 쉼표로 값을 구분합니다.

# 리스트 만들기
fruits = ['사과', '바나나', '딸기', '포도']
scores = [90, 85, 92, 78, 88]

print(fruits)
print(scores)
실행 결과
['사과', '바나나', '딸기', '포도'] [90, 85, 92, 78, 88]
개념 인덱스(Index) — 원소 꺼내기

리스트의 각 원소는 인덱스(순서 번호)로 꺼낼 수 있습니다.
파이썬에서 인덱스는 0부터 시작합니다!

fruits = ['사과', '바나나', '딸기', '포도']
#          [0]      [1]      [2]    [3]

print(fruits[0])  # 첫 번째 원소
print(fruits[2])  # 세 번째 원소
print(fruits[-1]) # 마지막 원소 (-1은 뒤에서 첫 번째)
실행 결과
사과 딸기 포도

🔨 따라하기

1
리스트 길이 구하기
scores = [90, 85, 92, 78, 88]

print('개수:', len(scores))    # len() = 길이
print('최고점:', max(scores))  # max() = 최댓값
print('최저점:', min(scores))  # min() = 최솟값
실행 결과
개수: 5 최고점: 92 최저점: 78
💡 나중에 지도에서도 쓰여요!
지도에 마커를 찍을 때도 [위도, 경도] 형태의 리스트를 써요.
예) location=[37.5, 127.0] — 서울 중심 좌표

⚠️ 주의할 점

인덱스 범위를 벗어나면 오류!
fruits = ["사과", "바나나"]에서 fruits[5]를 하면 IndexError가 납니다.
리스트 길이보다 큰 인덱스는 사용할 수 없습니다.
