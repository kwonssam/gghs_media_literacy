📌 데이터 불러오기

구글 드라이브에서 CSV 파일을 자동으로 찾아서 불러옵니다.
glob은 파일을 자동으로 검색해주는 도구예요.

2
CSV 파일 불러오기
Python
import pandas as pd
import glob

# 드라이브 전체에서 파일 자동 검색
files = glob.glob('/content/drive/MyDrive/**/KR_유튜브_트렌딩.csv', recursive=True)
df = pd.read_csv(files[0], encoding='utf-8-sig')
df
💡 pandas란?
파이썬에서 표(데이터)를 다루는 라이브러리예요.
pd는 pandas의 별명이에요. df는 DataFrame의 약자로, 표 형태의 데이터를 담는 변수예요.

📌 데이터 탐색하기

3
데이터 크기 확인 — df.shape

이 데이터가 몇 행, 몇 열인지 확인합니다.

Python
df.shape
실행 결과
(262100, 9)
결과 읽는 법: (행 수, 열 수) 로 나타납니다.
262,100행 × 9열 = 약 26만 개의 유튜브 트렌딩 데이터!
4
처음 5행만 보기 — df.head()

데이터가 너무 많아서 전체를 보기 어려울 때, 처음 몇 행만 골라서 봅니다.

Python
df.head()       # 처음 5행
df.head(10)    # 처음 10행
📝 직접 해봐요: 괄호 안 숫자를 바꿔보세요. 몇 행이 나오나요?
5
데이터 정보 보기 — df.info()

각 컬럼의 이름, 데이터 타입, 결측값 개수를 한 번에 확인합니다.

Python
df.info()
실행 결과 예시
RangeIndex: 262100 entries, 0 to 262099 Data columns (total 9 columns): # Column Non-Null Count Dtype --- ------ -------------- ----- 0 제목 262100 non-null object 1 채널명 262100 non-null object 2 카테고리 262100 non-null object ...
💡 결측값이란? 비어 있는 데이터예요.
Non-Null Count가 전체 행 수보다 작으면 결측값이 있는 거예요.
6
컬럼 이름 확인 — df.columns

어떤 항목(컬럼)이 있는지 목록을 봅니다.

Python
df.columns
실행 결과

특정 컬럼만 선택하기

Python
# 채널명 컬럼만 보기
df['채널명']

예) df['카테고리'], df['조회수']

📌 값 개수 세기 — value_counts()

각 값이 몇 번 등장하는지 세어줍니다. 많이 등장한 순서로 정렬됩니다.

8
카테고리별 영상 수
Python
df['카테고리'].value_counts()
실행 결과 예시
엔터테인먼트 89423 음악 54821 게임 32104 뉴스&정치 28903 ...
9
상위 몇 개만 보기 — .head()

.head(숫자)를 붙이면 상위 몇 개만 볼 수 있습니다.

Python
# 카테고리 TOP 5
df['카테고리'].value_counts().head(5)

# 트렌딩에 가장 많이 오른 채널 TOP 10
df['채널명'].value_counts().head(10)
📝 직접 해봐요: 숫자를 바꿔서 TOP 3, TOP 20도 확인해봐요.
10
연도별 영상 수 — sort_index()

연도 순서대로 정렬해서 보려면 .sort_index()를 붙여야 해요.

Python
df['연도'].value_counts().sort_index()
💡 value_counts()는 많은 순서로, sort_index()는 연도 순서로 정렬해요.
다음 차시에서 이 결과로 선그래프를 그려볼 거예요!
11
숫자 데이터 요약 통계 — df.describe()

숫자 컬럼들의 최솟값, 최댓값, 평균 등을 한 번에 확인합니다.

Python
df.describe()
실행 결과 예시
조회수 좋아요 count 2.621e+05 2.621e+05 mean 3.241e+06 1.203e+05 min 0.000e+00 0.000e+00 max 4.371e+08 1.421e+07 ...
💡 각 행의 의미:
count 개수 / mean 평균 / min 최솟값 / max 최댓값

📋 오늘 배운 함수 정리

코드하는 일
df.shape행 수, 열 수 확인
df.head()처음 5행 보기
df.info()컬럼 정보, 타입, 결측값 확인
df.columns컬럼 이름 목록
df['컬럼명']특정 컬럼만 선택
df['컬럼명'].value_counts()값 개수 세기 (많은 순)
df.describe()숫자 컬럼 요약 통계

🐾 두 번째 데이터 — 고양시 동물병원

같은 방법으로 고양시 동물병원 데이터도 탐색해봐요.
코드 구조는 유튜브 데이터와 똑같아요. 파일명과 인코딩만 달라요.

12
동물병원 데이터 불러오기
Python
# 고양시 동물병원 데이터 불러오기
files2 = glob.glob('/content/drive/MyDrive/**/경기도_고양시_동물병원현황.csv', recursive=True)
df2 = pd.read_csv(files2[0], encoding='cp949')

# 주소에서 동명 추출 — 괄호 안 텍스트를 꺼내요
# 예) '경기도 고양시 덕양구 호국로 778-6 (성사동)' → '성사동'
df2['동명'] = df2['사업장소재지(도로명)'].str.extract(r'\(([^)]+)\)')

df2.head()
💡 이 데이터는 인코딩이 cp949예요.
한글 파일은 utf-8-sig 또는 cp949 중 하나로 돼있어요. 오류가 나면 바꿔봐요.
13
동물병원 데이터 탐색하기

유튜브 데이터와 동일한 함수로 탐색해봐요.

Python
# 데이터 크기
print(df2.shape)

# 컬럼 확인
print(df2.columns.tolist())

# 동별 병원 수
df2['동명'].value_counts()
실행 결과 예시
(124, 4) ['업소명', '사업장소재지(도로명)', '전화번호', '동명'] 동명 장항동 9 행신동 8 대화동 8 ...
📝 직접 해봐요: 병원이 가장 많은 동과 가장 적은 동은 어디인가요?
왜 그럴까요? 이유를 생각해봐요.
