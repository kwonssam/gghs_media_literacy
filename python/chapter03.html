3차시

숫자를 그래프로 —
데이터 시각화

지난 시간에 리스트와 데이터 탐색 방법을 배웠어요.
이번엔 그 숫자들을 그래프로 변환합니다.
막대, 선, 점, 원 — 네 가지 그래프를 직접 그려보고 옵션도 바꿔봐요.

🎯 이 장에서 배우는 것

학습 목표
  • 데이터의 특성에 맞는 그래프 종류를 고를 수 있다
  • .plot(kind=...)으로 막대·선·점·원 그래프를 그릴 수 있다
  • title, figsize, color 등 옵션을 조정할 수 있다
  • 유튜브 트렌딩 데이터로 "어떤 채널이 제일 많이 떴나?"를 시각화할 수 있다

📌 핵심 개념 — 어떤 그래프를 써야 할까?

그래프는 아무 거나 쓰는 게 아니에요. 데이터의 특성에 따라 적합한 종류가 달라요. 아래 기준을 외워두면 수행평가에서도 바로 써먹을 수 있어요.

📊
막대그래프 (bar)
항목끼리 크기를 비교할 때
예) 카테고리별 영상 수, 채널별 트렌딩 횟수
📈
선그래프 (line)
시간의 흐름에 따른 변화를 볼 때
예) 연도별 트렌딩 영상 수, 월별 추이
🔵
점그래프 (scatter)
두 수치 사이의 관계를 볼 때
예) 조회수 vs 좋아요 수
🥧
원그래프 (pie)
전체 중 각 항목의 비율을 볼 때
예) 카테고리별 비율, 성별 비율

아래 두 줄은 항상 맨 위에서 실행해요. 한글 폰트와 pandas를 불러오는 코드예요.

📊 막대그래프

개념 막대그래프 — .plot(kind='bar')

1차시에서 value_counts()로 카테고리별 영상 수를 숫자 표로 봤어요.
이번엔 그 결과를 그대로 `.plot()`에 넘기면 막대그래프가 돼요.

Python
df['카테고리'].value_counts().plot(kind='bar')
실행 결과 (이미지)
엔터테인먼트, 음악, 게임, 뉴스&정치 … 순서로 막대가 그려짐

🔨 따라하기 — 옵션 하나씩 추가하기

1
기본 막대그래프 그리기

가장 간단한 코드예요. 카테고리별 트렌딩 영상 수를 막대로 그려요.

df['카테고리'].value_counts().plot(kind='bar')
2
제목, 크기, 색깔 추가하기

그래프가 너무 작거나 밋밋하면 아래 옵션을 추가해봐요.

df['카테고리'].value_counts().plot(
    kind='bar',
    title='카테고리별 트렌딩 영상 수',  # 제목
    figsize=(12, 5),               # 그래프 크기 (가로, 세로)
    color='steelblue',              # 색깔
    rot=30                           # x축 글자 기울기 (도)
)
3
채널별 TOP 10 뽑아서 그리기

모든 채널을 다 그리면 너무 많아요. .head(10)으로 상위 10개만 골라봐요.

df['채널명'].value_counts().head(10).plot(
    kind='bar',
    title='트렌딩 횟수 TOP 10 채널',
    figsize=(12, 5),
    color='coral',
    edgecolor='black',               # 막대 테두리
    width=0.6,                        # 막대 두께 (0~1)
    rot=30
)
결과 읽기: 파뿌리? 숏박스? 어떤 채널이 1등인가요? 실제 데이터로 확인해봐요!

🎨 자주 쓰는 옵션 모음

옵션역할예시 값
title그래프 제목'제목입력'
figsize그래프 크기 (가로, 세로)(12, 5)
color막대 색깔'steelblue', 'coral', '#3b82f6'
rotx축 글자 기울기 (도)0, 30, 45, 90
width막대 두께0.4 ~ 0.9
edgecolor막대 테두리 색'black', 'white'

🐾 고양시 동물병원 데이터 — 막대그래프

코드 구조는 유튜브와 완전히 동일해요. df → df2, 컬럼명만 바꾸면 돼요!

STEP 1 — 기본 막대그래프

df2['동명'].value_counts().plot(kind='bar')

STEP 2 — 옵션 적용하기

df2['동명'].value_counts().head(10).plot(
    kind='bar',
    title='고양시 동별 동물병원 수 TOP 10',
    figsize=(12, 5),
    color='mediumseagreen',   # ← 바꿔보기
    rot=30,
    edgecolor='black'
)

STEP 3 — 가로 막대그래프 (barh)

df2['동명'].value_counts().head(10).plot(
    kind='barh',
    title='고양시 동별 동물병원 수 TOP 10 (가로)',
    figsize=(10, 6),
    color='mediumseagreen'
)

📈 선그래프 — 연도별 트렌딩이 어떻게 바뀌었을까?

개념 선그래프 — .plot(kind='line')

시간에 따른 변화를 보고 싶을 때 사용해요.
value_counts() 뒤에 .sort_index()를 붙이면 연도 순서대로 정렬돼요.

Python
df['연도'].value_counts().sort_index().plot(kind='line')

🔨 따라하기

1
연도별 트렌딩 영상 수 변화 보기
df['연도'].value_counts().sort_index().plot(
    kind='line',
    title='연도별 트렌딩 영상 수',
    figsize=(10, 5),
    color='green',
    linewidth=3,    # 선 두께
    marker='o'      # 데이터 포인트 모양 (o, s, ^, *)
)
2
선 스타일 바꿔보기

linestyle 옵션으로 선 모양을 바꿀 수 있어요.

df['연도'].value_counts().sort_index().plot(
    kind='line',
    title='연도별 트렌딩 영상 수',
    figsize=(10, 5),
    color='tomato',
    linewidth=2,
    linestyle='--',   # 선 모양: - | -- | : | -.
    marker='s',       # 네모 포인트
    markersize=10     # 포인트 크기
)

🎨 선그래프 주요 옵션

옵션역할예시 값
linewidth선 두께1, 2, 3
linestyle선 모양'-' '--' ':' '-.'
marker데이터 포인트 모양'o' 's' '^' '*'
markersize포인트 크기5, 10

🐾 고양시 동물병원 데이터 — 선그래프

동물병원 데이터는 선그래프에 적합하지 않아요.
선그래프는 시간에 따른 변화를 보여주는 그래프예요.
동물병원 데이터에는 날짜/연도 컬럼이 없어서 선그래프를 그리기 어려워요.
→ 동물병원 데이터는 막대그래프와 원그래프가 더 적합해요!

🔵 점그래프 — 조회수와 좋아요는 관계가 있을까?

개념 점그래프 — .plot(kind='scatter')

점그래프는 두 수치 컬럼을 x축과 y축에 각각 지정해야 해요.
점이 오른쪽 위로 모이면 두 값이 비례관계, 흩어져 있으면 관련이 적다는 뜻이에요.

Python
df.plot(kind='scatter', x='조회수', y='좋아요')

🔨 따라하기

1
조회수 vs 좋아요 점그래프
df.plot(
    kind='scatter',
    x='조회수', y='좋아요',
    title='조회수 vs 좋아요',
    figsize=(10, 6),
    color='purple',
    alpha=0.3,    # 투명도 (0~1, 점이 겹칠 때 유용)
    s=10          # 점 크기
)
결과 읽기: 점들이 오른쪽 위로 몰려 있나요? 그렇다면 조회수가 많을수록 좋아요도 많은 경향이 있는 거예요.
2
조회수 vs 댓글수 비교해보기

y축을 '댓글수'로 바꾸면 조회수-댓글 관계도 볼 수 있어요.

df.plot(
    kind='scatter',
    x='조회수', y='댓글수',   # y축만 바꿈
    title='조회수 vs 댓글수',
    figsize=(10, 6),
    color='teal',
    alpha=0.3,
    s=10
)
⚠️ alpha 옵션이 중요한 이유
유튜브 데이터는 26만 행이에요. 점이 너무 많아 겹치면 분포를 알 수 없어요.
alpha=0.1~0.3 정도로 투명하게 만들면 밀집된 곳이 진하게 보여서 훨씬 잘 읽혀요.

🐾 고양시 동물병원 데이터 — 점그래프

동물병원 데이터는 점그래프에 적합하지 않아요.
점그래프는 두 수치 컬럼의 관계를 보여주는 그래프예요.
동물병원 데이터에는 비교할 수치 컬럼(조회수, 좋아요 등)이 없어요.
→ 수치 데이터가 있는 유튜브 트렌딩 데이터에 더 잘 맞아요!

🥧 원그래프 — 카테고리 비율은 어떻게 될까?

개념 원그래프 — .plot(kind='pie')

전체를 100%로 놓고 각 항목이 차지하는 비율을 보여줘요.
항목이 너무 많으면 조각이 너무 작아져요. .head(7)로 상위 7개만 보여주는 게 깔끔해요.

Python
df['카테고리'].value_counts().head(7).plot(kind='pie')

🔨 따라하기

1
퍼센트 표시 + 시작 각도 설정
df['카테고리'].value_counts().head(7).plot(
    kind='pie',
    title='카테고리별 비율 TOP 7',
    figsize=(8, 8),
    autopct='%1.1f%%',   # 퍼센트 표시 (소수점 1자리)
    startangle=90       # 12시 방향부터 시작
)
2
1등 조각 강조하기 (explode)

explode 옵션으로 특정 조각을 살짝 떼어낼 수 있어요. 1등 카테고리를 강조할 때 써봐요.

df['카테고리'].value_counts().head(7).plot(
    kind='pie',
    title='카테고리별 비율 TOP 7',
    figsize=(8, 8),
    autopct='%1.1f%%',
    startangle=90,
    explode=[0.1, 0, 0, 0, 0, 0, 0]  # 첫 번째 조각(1등)만 0.1 떼어냄
)
주의: explode 리스트의 개수는 .head(숫자)와 꼭 같아야 해요.
.head(7)이면 explode도 7개짜리 리스트여야 해요.

🐾 고양시 동물병원 데이터 — 원그래프

동별 병원 비율을 원그래프로 확인해봐요.

STEP 1 — 기본 원그래프

df2['동명'].value_counts().head(7).plot(kind='pie')

STEP 2 — 옵션 적용하기

df2['동명'].value_counts().head(7).plot(
    kind='pie',
    title='고양시 동별 동물병원 비율',
    figsize=(8, 8),
    autopct='%1.1f%%',        # 퍼센트 표시
    startangle=90             # 시작 각도
)

STEP 3 — 1등 동 강조하기 (explode)

df2['동명'].value_counts().head(7).plot(
    kind='pie',
    title='고양시 동별 동물병원 비율 — 1등 강조',
    figsize=(8, 8),
    autopct='%1.1f%%',
    startangle=90,
    explode=[0.1, 0, 0, 0, 0, 0, 0]   # 1등 강조
)

✏️ 직접 해봐요

# TOP 5로 바꾸고, 2등 동을 강조해봐요!
df2['동명'].value_counts().head(   ).plot(
    kind='pie',
    title='',
    figsize=(7, 7),
    autopct='%1.1f%%',
    startangle=90,
    explode=[  ,   ,   ,   ,   ]   # 2등 위치만 0.1로 바꾸기
)

⚠️ 주의할 점

그래프가 안 나올 때
Jupyter/Colab에서는 보통 자동으로 그래프가 뜨지만, 가끔 안 나오는 경우가 있어요.
코드 마지막 줄에 아래를 추가해봐요. 
import matplotlib.pyplot as plt
plt.show()
x축 글자가 겹쳐서 안 보일 때
항목 이름이 길면 겹쳐 보여요. rot=45 또는 rot=90으로 기울여봐요. 
...plot(kind='bar', rot=45)
한글이 깨져서 네모로 나올 때
맨 위에 koreanize_matplotlib를 import했는지 확인해봐요. 빠뜨렸으면 다시 실행해야 해요.

    kind='pie',
    title='고양시 동별 동물병원 비율',

🎯 아래 항목을 모두 할 수 있으면 다음 차시로 넘어가세요!
  • .plot(kind='bar')로 막대그래프를 그릴 수 있다
  • title, figsize, color, rot 옵션을 조정할 수 있다
  • .head(10)으로 상위 10개만 골라서 그릴 수 있다
  • .sort_index()가 왜 필요한지 설명할 수 있다
  • 점그래프에서 x, y 옵션이 뭘 의미하는지 안다
  • alpha 옵션이 왜 점그래프에서 중요한지 설명할 수 있다
  • 원그래프에서 autopct가 어떤 역할인지 안다
