📌 핵심 개념

개념 워드클라우드란?
텍스트에서 자주 등장하는 단어일수록 크게 표시하는 시각화예요.
뉴스 기사, SNS 댓글, 영상 제목 등 텍스트 데이터라면 무엇이든 만들 수 있어요.
개념 언제 쓰나요?
숫자로 표현하기 어려운 텍스트 데이터의 패턴을 볼 때 써요.
"요즘 유튜브에서 어떤 주제가 핫할까?" 같은 질문에 답할 수 있어요.
개념 불용어(Stopword)란?
"이", "가", "을", "은", "의" 처럼 의미 없이 자주 나오는 단어예요.
제거하지 않으면 워드클라우드가 이런 단어들로 가득 차요.
개념 오늘 쓸 라이브러리
wordcloud — 워드클라우드 그리기
matplotlib — 그림 출력하기
pandas — 데이터 불러오기 (1~2차시에서 썼던 것)
💡 엑셀로는 못 하는 것
막대·선·원 그래프는 엑셀로도 할 수 있어요. 하지만 워드클라우드는 파이썬으로만 가능해요.
텍스트 데이터 시각화는 파이썬의 강점 중 하나예요.

from wordcloud import WordCloud, ImageColorGenerator
import matplotlib.pyplot as plt
import numpy as np
from PIL import Image
from collections import Counter

text = '''
더불어 함께 살아가는 세계시민 교육
협업 다른 사람과 책임 공유 현실적인 결정 상호 의존하여 활동
문제해결력 창의성 현실 적용 가능한 문제해결 다양한 방법으로 문제해결 혁신적인 해결방법
자기조절 책임감 스스로 기획 적용 피드백 장기적으로 책임감 있는 노력
지식구성 지식을 구성하고 적용 여러 학문분야 지식구성 경험 공유
글로벌 의식 세계 문제 세계인과 함께 문제해결 외국어 구사
커뮤니케이션 소통 학교문화 세계인과 소통하는 국제교육 진정성 있는 소통
창의성과 인성을 갖추고 세계와 소통하는 글로벌 리더 육성
배움의 열정과 나눔의 행복이 넘치는 학교
'''

# split() — 텍스트를 공백 기준으로 단어 리스트로 나눠요
# '문제해결 창의성 소통' → ['문제해결', '창의성', '소통']
word_count = Counter(text.split())
word_count.most_common(10)
실행 결과 예시
[('문제해결', 4), ('소통', 3), ('세계인과', 2), ('책임감', 2), ('지식구성', 2), ...]
💡 split() vs Counter()
split() — 텍스트를 단어 리스트로 쪼개요. 'A B C'.split() → ['A', 'B', 'C']
Counter() — 리스트에서 각 항목이 몇 번 나왔는지 세요. Counter(['A','A','B']) → {'A':2, 'B':1}
2
기본 워드클라우드 + 옵션 바꾸기

generate_from_frequencies()는 이미 계산된 빈도(Counter)를 넣어서 워드클라우드를 만드는 방법이에요.

font_path = 'NanumGothic'

wordcloud = WordCloud(
    font_path=font_path,
    width=800,
    height=400,
    background_color='white'
).generate_from_frequencies(word_count)
# generate_from_frequencies(빈도) — Counter 결과를 바로 넣어요
# cf. generate(text) — 텍스트를 넣으면 알아서 빈도를 계산해요

plt.figure(figsize=(12, 6))
plt.imshow(wordcloud, interpolation='bilinear')
plt.axis('off')
plt.show()

⚠️ PNG 파일 필요
이미지 파일은 PNG 형식이어야 해요. gghs.png 파일을 먼저 업로드하세요. 
# GGHS 로고 이미지 업로드
from google.colab import files
uploaded = files.upload()   # gghs.png 파일 선택
# Image.open() — PNG 파일을 열어요
# np.array() — 이미지를 숫자 배열로 변환해요 (WordCloud가 이해하는 형태)
mask = np.array(Image.open('gghs.png'))

wordcloud_mask = WordCloud(
    font_path=font_path,
    mask=mask,              # ← 마스크 적용! 이 한 줄이 핵심이에요
    background_color='white'
).generate_from_frequencies(word_count)

plt.figure(figsize=(10, 10))
plt.imshow(wordcloud_mask, interpolation='bilinear')
plt.axis('off')
plt.show()
실행 결과
GGHS 로고 모양 안에 교육목표 단어들이 가득 채워진 워드클라우드가 나타나요! 로고 바깥 영역은 비어 있고, 안쪽에만 단어가 들어가요.
💡 mask란?
이미지의 하얀 부분 = 단어가 들어가는 곳
이미지의 어두운 부분 = 단어가 들어가지 않는 곳
배경이 투명한 PNG 파일을 쓰면 가장 깔끔하게 나와요.
4
이미지의 색깔을 단어 색깔로 활용하기

ImageColorGenerator를 사용하면 마스크 이미지의 색깔이 각 단어의 색깔로 적용돼요.

wordcloud_color = WordCloud(
    font_path=font_path,
    mask=mask,
    background_color='white'
).generate_from_frequencies(word_count)

# ImageColorGenerator — 마스크 이미지에서 색깔 정보를 추출해요
image_colors = ImageColorGenerator(mask)

plt.figure(figsize=(10, 10))
# .recolor(color_func=image_colors) — 단어 색깔을 이미지 색깔로 교체해요
plt.imshow(wordcloud_color.recolor(color_func=image_colors),
           interpolation='bilinear')
plt.axis('off')
plt.show()

# 제목 컬럼의 빈 값(NaN)을 제거하고, 모든 제목을 공백으로 이어 붙여요
# ['BTS 신곡', '먹방 브이로그'] → 'BTS 신곡 먹방 브이로그'  처럼 하나의 긴 텍스트가 돼요
text = ' '.join(df['제목'].dropna())

# WordCloud 객체를 만들어요 — 가로 800px, 세로 400px, 흰 배경
# .generate(text)로 단어 빈도를 분석해서 워드클라우드 이미지를 생성해요
wc = WordCloud(
    width=800,
    height=400,
    background_color='white'
).generate(text)

# 그림 캔버스 크기를 설정해요 — figsize=(가로, 세로) 단위는 인치
plt.figure(figsize=(14, 7))
# 워드클라우드 이미지를 캔버스에 올려요 — bilinear는 이미지를 부드럽게 보이게 해요
plt.imshow(wc, interpolation='bilinear')
# x축·y축 눈금을 숨겨요 — 워드클라우드는 그래프가 아닌 그림이라 눈금이 필요 없어요
plt.axis('off')
# 제목을 달고 화면에 표시해요 — plt.show()가 없으면 아무것도 안 나와요!
plt.title('유튜브 트렌딩 제목 워드클라우드')
plt.show()

흰 배경에 단어들이 크기별로 흩어진 구름 모양 그림이 나타나요. 자주 등장한 단어(예: BTS, 먹방, 브이로그)가 크게, 드물게 나온 단어가 작게 보여요.

# 아래 값들을 하나씩 바꿔가며 결과가 어떻게 달라지는지 확인해봐요!
wc = WordCloud(
    width=1000,                   # 이미지 가로 크기 (픽셀)
    height=500,                  # 이미지 세로 크기 (픽셀)
    background_color='black',    # ← 바꿔보기: 'white', 'navy'
    max_words=100,               # ← 바꿔보기: 50, 150, 200
    max_font_size=120,           # ← 바꿔보기: 80, 200
    colormap='plasma'            # ← 바꿔보기: 'viridis', 'Set2', 'Reds'
).generate(text)

plt.figure(figsize=(14, 7))
plt.imshow(wc, interpolation='bilinear')
plt.axis('off')
plt.show()
📝 직접 해봐요: colormap을 'viridis' → 'plasma' → 'Set2' 순서로 바꿔가며 색깔이 어떻게 달라지는지 확인해봐요!
