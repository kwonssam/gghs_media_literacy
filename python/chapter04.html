1
텍스트 입력 + 단어 빈도 확인

split()으로 단어를 나누고, Counter()로 빈도를 세봐요.

함수역할예시
text.split()공백 기준으로 단어 리스트로 나눔'소통 협업 소통' → ['소통','협업','소통']
Counter(리스트)각 항목이 몇 번 나왔는지 셈Counter([...]) → {'소통':2, '협업':1}
.most_common(10)많이 나온 순서로 TOP N 반환[('소통',2), ('협업',1), ...]
text = '''
더불어 함께 살아가는 세계시민 교육
협업 다른 사람과 책임 공유 현실적인 결정 상호 의존하여 활동
문제해결력 창의성 현실 적용 가능한 문제해결 다양한 방법으로 문제해결 혁신적인 해결방법
자기조절 책임감 스스로 기획 적용 피드백 장기적으로 책임감 있는 노력
지식구성 지식을 구성하고 적용 여러 학문분야 지식구성 경험 공유
글로벌 의식 세계 문제 세계인과 함께 문제해결 외국어 구사
커뮤니케이션 소통 학교문화 세계인과 소통하는 국제교육 진정성 있는 소통
창의성과 인성을 갖추고 세계와 소통하는 글로벌 리더 육성
배움의 열정과 나눔의 행복이 넘치는 학교
'''

word_count = Counter(text.split())
word_count.most_common(10)
실행 결과 예시
[('문제해결', 4), ('소통', 3), ('세계인과', 2), ('책임감', 2), ...]

cf. generate(text)는 텍스트를 넣으면 알아서 빈도를 계산해요. 

wordcloud = WordCloud(

GGHS 로고 마스크 적용하기

함수/옵션역할

5
이미지 색깔을 단어 색깔로 활용하기

ImageColorGenerator를 쓰면 마스크 이미지의 색깔이 각 단어의 색깔로 적용돼요.

방법단어 색깔핵심 코드
STEP 4 (마스크만)colormap 테마 색깔mask=mask
STEP 5 (이미지 색 활용)이미지 원래 색깔.recolor(color_func=image_colors)
wordcloud_color = WordCloud(
    font_path=font_path,
    mask=mask,
    background_color='white'
).generate_from_frequencies(word_count)

# ImageColorGenerator — 마스크 이미지에서 색깔 정보 추출
image_colors = ImageColorGenerator(mask)

plt.figure(figsize=(10, 10))
# .recolor() — 단어 색깔을 이미지 색깔로 교체
plt.imshow(wordcloud_color.recolor(color_func=image_colors),
           interpolation='bilinear')
plt.axis('off')
plt.show()
📝 직접 해봐요: STEP 4와 STEP 5 결과를 비교해봐요. 색깔이 어떻게 달라지나요?

📋 불용어 찾기 프롬프트

# ── 준비 ──────────────────────────────────────────

import pandas as pd
from wordcloud import WordCloud
import matplotlib.pyplot as plt
import glob

files = glob.glob('/content/drive/MyDrive/**/KR_유튜브_트렌딩.csv', recursive=True)
df = pd.read_csv(files[0], encoding='utf-8-sig')

# ── 텍스트 합치기 ─────────────────────────────────
text = ' '.join(df['제목'].dropna())

# ── 불용어 설정 ───────────────────────────────────
stopwords = {'이', '가', '을', '는', '은', '의', '에', '로',
             '하다', '있다', 'EP', 'MV', 'TV'}

# ── 워드클라우드 만들기 ───────────────────────────
wc = WordCloud(
    width=1000,
    height=500,
    background_color='white',
    max_words=100,
    colormap='viridis',
    stopwords=stopwords
).generate(text)

# ── 출력 ─────────────────────────────────────────
plt.figure(figsize=(14, 7))
plt.imshow(wc, interpolation='bilinear')
plt.axis('off')
plt.title('유튜브 트렌딩 제목 워드클라우드')
plt.show()

⚠️ 주의할 점

한글이 네모(□)로 나올 때
wordcloud가 한글 폰트를 못 찾는 거예요. LLM에게 이렇게 물어봐요:
"구글 코랩에서 한글 폰트를 설치하고 wordcloud에 적용하는 코드 알려줘"
generate() 오류 — "no words" 메시지
텍스트가 비어 있거나 모든 단어가 불용어에 걸린 경우예요.
print(len(text))로 텍스트 길이를 먼저 확인해봐요.
워드클라우드는 한국어 형태소 분석이 없어요
