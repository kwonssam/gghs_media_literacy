📌 리스틀리로 크롤링하기

리스틀리는 코딩 없이 클릭만으로 웹 데이터를 수집하는 도구예요.
구글 맵스에서 원하는 장소의 리뷰를 자동으로 가져올 수 있어요.

1

리뷰를 수집할 장소를 먼저 정해요. 리뷰가 많을수록 워드클라우드 결과가 풍성해요.

💡 장소 선택 팁:
리뷰가 100개 이상인 곳을 고르는 게 좋아요.
예) 유명 맛집, 카페, 관광지, 영화관 등
2
리스틀리로 크롤링하기

선생님이 알려주는 순서대로 리스틀리를 실행해봐요.

순서내용
①리스틀리 접속 후 로그인
②구글 맵스에서 선택한 장소 검색
③리뷰 탭 클릭 → 리스틀리로 크롤링 시작
④엑셀(.xlsx) 파일로 다운로드
⑤다운받은 파일을 구글 드라이브에 업로드
⚠️ 주의: 크롤링이 완료되면 반드시 파일을 구글 드라이브에 업로드해야 해요.
파일명에 한글이나 띄어쓰기가 없도록 저장하는 게 좋아요.

📌 크롤링 데이터 불러오기

3
xlsx 파일 불러오기

리스틀리로 크롤링한 파일은 xlsx(엑셀) 형식이에요.
CSV와 달리 pd.read_excel()로 불러와요.

Python
import pandas as pd
import glob

# 드라이브에서 xlsx 파일 자동 검색
files = glob.glob('/content/drive/MyDrive/**/*.xlsx', recursive=True)
df = pd.read_excel(files[0])
df.head()
💡 CSV vs xlsx:
pd.read_csv() → CSV 파일 (.csv)
pd.read_excel() → 엑셀 파일 (.xlsx)
4
리뷰 텍스트 컬럼 확인하기

리스틀리 크롤링 결과는 컬럼명이 LABEL-1, LABEL-2... 형태예요.
리뷰 텍스트는 LABEL-11 컬럼에 있어요.

Python
# 컬럼 확인
print(df.columns.tolist())

# LABEL-11이 리뷰 텍스트예요
df['LABEL-11'].head(5)
실행 결과 예시
0 남편과 건강검진을 받았습니다. 대부분의 병원들은 검진후 엑스레이나... 1 친절하고 시설이 깔끔해요. 대기시간이 조금 길지만 의사 선생님이... 2 주차가 편리하고 접근성이 좋아요. 다음에도 방문할 것 같습니다... ...
📝 직접 해봐요: df.shape으로 몇 개의 리뷰가 수집됐는지 확인해봐요.

📌 리뷰 워드클라우드 만들기

5
텍스트 합치고 워드클라우드 그리기

4차시에서 배운 방법 그대로예요. 데이터만 달라졌어요!

Python
from wordcloud import WordCloud
import matplotlib.pyplot as plt
from collections import Counter

# 리뷰 텍스트 합치기 — 빈 값 제거 후 하나로
text = ' '.join(df['LABEL-11'].dropna().astype('str'))

# 자주 나온 단어 TOP 30 확인
word_counts = Counter(text.split())
top30 = word_counts.most_common(30)
print(top30)
💡 .astype('str')이란?
리뷰 텍스트가 가끔 숫자로 저장돼 있을 수 있어요.
.astype('str')로 모두 문자열로 변환한 뒤 합쳐요.
6
불용어 제거 + 워드클라우드 출력

TOP 30 결과를 보고 LLM에게 불용어를 골라달라고 요청한 뒤, 워드클라우드를 완성해요.

📋 불용어 찾기 프롬프트 (4차시와 동일) 
아래는 구글 맵스 리뷰에서 자주 나온 단어 TOP 30이야.
워드클라우드에서 의미 없이 크게 나오는 불용어만 골라줘.
조사(이, 가, 을, 는), 너무 일반적인 단어들 위주로.
파이썬 집합(set) 형태로 바로 쓸 수 있게 출력해줘.

[여기에 top30 출력 결과 붙여넣기]
Python
# LLM이 준 불용어 목록으로 교체해요!
stopwords = {'이', '가', '을', '는', '은', '의', '에', '로', '도', '와'}

wc = WordCloud(
    font_path='NanumGothic',
    width=1000, height=500,
    background_color='white',
    max_words=100,
    colormap='viridis',
    stopwords=stopwords
).generate(text)

plt.figure(figsize=(14, 7))
plt.imshow(wc, interpolation='bilinear')
plt.axis('off')
plt.title('구글 맵스 리뷰 워드클라우드')
plt.show()
실행 결과
선택한 장소 리뷰에서 자주 나온 단어들이 구름 모양으로 표시돼요. 긍정적인 단어가 많으면 어떤 것들이 보이나요?
