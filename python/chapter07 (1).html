📌 보고서 작성 가이드

보고서는 구글 독스로 작성해요. 아래 항목을 순서대로 채워주세요.
① 선택 항목부터 시작해요. A 또는 B 중 하나를 고르세요.

①
연계 유형 선택 — A 또는 B 중 하나 선택
A. 교과 연계
  • 연계 교과목을 쓰세요.
  • 연계 단원명을 쓰세요.
  • 그 단원에서 배운 핵심 내용을 한 줄로 요약하세요.
  • 그 수업을 들으면서 내가 직접 경험하거나 느꼈던 것이 있나요? (뉴스에서 본 것, 주변에서 목격한 것, 의아하게 느꼈던 것 등)
  • 그 경험이나 느낌이 이번 데이터 분석 주제와 어떻게 연결되나요?
B. 사회 문제 연계
  • 관심 있는 사회 문제를 쓰세요.
  • 이 문제를 처음 알게 된 계기가 무엇인가요? (뉴스, 경험, 수업 등)
  • 이 문제가 나 또는 내 주변과 어떻게 연결되나요?
  • 데이터로 이 문제의 어떤 부분을 확인하고 싶나요?
②
분석 주제
  • 분석 주제를 한 문장으로 쓰세요.
  • 이 주제를 선택한 이유를 쓰세요.
③
데이터
  • 사용한 데이터의 이름을 쓰세요.
  • 데이터 출처(URL 포함)를 쓰세요.
  • 여러 데이터 중 이 데이터를 선택한 이유를 쓰세요.
  • 이 데이터의 주요 컬럼(항목)을 3가지 이상 쓰세요.
④
코랩 링크
  • 분석 코드가 담긴 구글 코랩 링크를 붙여넣으세요.
⑤
시각화 유형 선택 이유
  • 사용한 시각화 유형을 쓰세요. (막대 / 선 / 점 / 원 / 워드클라우드 / 지도)
  • 이 데이터에 이 유형이 적합한 이유를 쓰세요.
  • 다른 유형을 선택하지 않은 이유를 쓰세요.
⑥
분석 결과
  • 그래프에서 가장 눈에 띄는 점을 쓰세요.
  • 교과에서 배운 내용 또는 예상과 일치한 점은 무엇인가요?
  • 교과에서 배운 내용 또는 예상과 달랐던 점은 무엇인가요?
  • 달랐다면 그 이유가 무엇이라고 생각하나요?
⑦
데이터의 한계
  • 이 데이터에 포함되지 않은 정보가 있다면 무엇인가요?
  • 그 정보가 있었다면 분석 결과가 어떻게 달라졌을까요?

📌 개별 확인 안내

개별 확인 방식
  • 10차시 이후 수업 중 선생님이 한 명씩 개별 확인해요
  • 코드 설명 손글씨 종이를 보면서 코드 각 부분을 직접 설명해요
  • "이 코드 어떻게 동작해요?", "왜 이 그래프를 선택했어요?" 같은 질문에 답해요
  • 대본 없이 자신의 언어로 설명해야 해요
📝 확인 중에 묻는 질문 예시
· 이 코드 각 부분이 뭘 하는지 설명해봐요
· 왜 이 시각화 방법을 선택했어요?
· 그래프 결과를 보고 어떤 점이 가장 인상적이었어요?
· 이 데이터의 한계가 뭐라고 생각해요?
💡 코드 설명 작성 팁
단순히 "리스트를 만든다"가 아니라 "왜 이렇게 짰는지"까지 써야 해요.
예) "카테고리별로 분류하기 위해 딕셔너리 대신 리스트를 사용했어요."
