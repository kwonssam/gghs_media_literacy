PART 0 — 파일 3개 만들기

윈도우는 메모장, 맥북은 TextEdit(텍스트 편집기)를 써요. 아래에서 내 컴퓨터에 맞는 탭을 눌러요.

📁 game/ ← 바탕화면에 이 폴더 만들기 ├── index.html ← 뼈대 ├── style.css ← 꾸미기 └── script.js ← 동작
파일 이름 주의! index.HTML, Style.css 처럼 대소문자 틀리면 연결이 안 돼요. 정확하게 소문자로 써요.
🪟 윈도우 — 메모장
🍎 맥북 — TextEdit
윈도우 메모장으로 파일 만들기
1
바탕화면에 폴더 만들기

바탕화면 빈 곳에서 우클릭 → 새로 만들기 → 폴더 → 이름: game

2
메모장 열기

시작 버튼 → "메모장" 검색 → 열기

3
index.html 저장하기

메모장이 빈 상태에서 → 파일 → 다른 이름으로 저장

저장 위치: 방금 만든 game 폴더 파일 이름: index.html 파일 형식: 모든 파일 (*.*) ← 반드시 이걸로! 인코딩: UTF-8 → 저장
⚠️ 파일 형식을 "텍스트 문서"로 놔두면 index.html.txt 로 저장돼요!
4
style.css, script.js도 같은 방법으로 만들기

새 메모장 창 열고 → 파일 → 다른 이름으로 저장 → game 폴더에 저장

두 번째 파일: style.css 세 번째 파일: script.js (파일 형식: 모든 파일, 인코딩: UTF-8 동일)
5
game 폴더 확인

game 폴더 열어보면 파일 3개가 있어야 해요.

✅ 파일 아이콘이 브라우저 모양(index.html), 메모장 모양(style.css, script.js)으로 보이면 성공!
6
파일 열어서 코딩하기

파일에서 우클릭 → 연결 프로그램 → 메모장 → 코드 입력 → 저장

💡 index.html을 더블클릭하면 브라우저로 열려요. 코딩할 때는 꼭 우클릭 → 메모장으로 열어요!

PART 1 — index.html 만들기

HTML은 뼈대예요. 어디에 무엇이 있는지 구조를 잡고, CSS와 JS 파일을 연결해요.

index.html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>클릭 타겟 게임</title>

  <!-- CSS 파일 연결 -->
  <link rel="stylesheet" href="style.css">

</head>
<body>

  <h1>🎯 클릭 타겟 게임</h1>

  <!-- 점수판 -->
  <div id="scoreboard">
    <span>점수: <strong id="score">0</strong></span>
    <span>시간: <strong id="timer">10</strong>초</span>
  </div>

  <!-- 게임판 -->
  <div id="game-area"></div>

  <!-- 결과 메시지 -->
  <div id="result"></div>

  <!-- 시작 버튼 -->
  <button id="start-btn" onclick="startGame()">게임 시작</button>

  <!-- JS 파일 연결 — body 맨 끝에! -->
  <script src="script.js"></script>

</body>
</html>
id가 뭔가요? 이름표예요. JS에서 나중에 "score라는 id를 가진 요소를 찾아서 숫자를 바꿔라" 이렇게 써요.
script는 왜 body 맨 끝에? HTML이 다 만들어진 다음에 JS가 실행돼야 해요. 위에 있으면 HTML이 아직 없어서 오류가 나요.
  • 저장 후 index.html을 브라우저로 열었을 때 제목, 점수, 버튼이 보이나요?
  • 못생겨도 정상이에요 — CSS 아직 없어서 그래요!
PART 2 — style.css 만들기

CSS는 꾸미기예요. style.css 파일을 열고 아래 코드를 입력해요.

style.css
/* 전체 페이지 */
body {
  background: #1a1a2e;      /* 배경색 ← 바꿔보세요! */
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  height: 100vh;
  margin: 0;
  font-family: sans-serif;
  color: white;
}

/* 점수판 */
#scoreboard {
  display: flex;
  gap: 40px;
  margin-bottom: 20px;
  font-size: 22px;
}

/* 게임판 */
#game-area {
  width: 600px;
  height: 400px;
  background: #16213e;
  border: 3px solid #e94560;
  border-radius: 12px;
  position: relative;   /* 원 위치의 기준점! */
  overflow: hidden;
}

/* 원 (타겟) */
.target {
  border-radius: 50%;    /* 50% = 원, 0% = 사각형 */
  background: #e94560;  /* 원 색 ← 바꿔보세요! */
  position: absolute;
  cursor: pointer;
  transition: transform 0.1s;
}

.target:hover {
  transform: scale(1.1);
}

/* 버튼 */
#start-btn {
  margin-top: 20px;
  padding: 12px 40px;
  font-size: 18px;
  background: #e94560;
  color: white;
  border: none;
  border-radius: 8px;
  cursor: pointer;
}

/* 결과 메시지 */
#result {
  font-size: 24px;
  font-weight: bold;
  color: #e94560;
  margin-top: 12px;
  height: 36px;
}
position: relative vs absolute? 게임판(relative)이 기준, 원(absolute)이 그 기준 안에서 위치를 잡아요.
CSS 도전 과제
  • 배경색 바꿔보기 (#1a1a2e → 다른 색 코드)
  • border-radius를 0%로 바꾸면 어떻게 될까?
  • 게임판 크기 바꿔보기 (width, height 숫자 변경)

PART 3 — script.js 만들기

JS는 뇌예요. script.js 파일을 열고 단계별로 입력해요.

📌 변수명은 왜 영어로 쓰나요?

JavaScript는 사실 한글 변수명도 기술적으로 됩니다. let 점수 = 0; 이렇게 써도 실제로 작동해요.

근데 실제 개발자들은 영어로 써요. 이유가 있어요:

🌍 전세계 개발자가 다 영어로 써요
🤖 AI한테 물어볼 때 영어 변수명이 훨씬 잘 통해요
📚 나중에 더 배울 때 혼란이 없어요
🔧 일부 환경에서 한글이 깨지는 경우가 있어요

그래서 이 수업에서도 영어 변수명 + 옆에 주석으로 설명을 다는 방식으로 써요.
let score = 0;   // 점수  ← 이렇게요!

3-1. 변수 선언 — 데이터를 담는 상자

script.js
// 변수 = 데이터를 담는 상자
let 점수 = 0;        // 현재 점수
let 남은시간 = 10;   // 10초 타이머
let 게임중 = false;  // 게임 진행 중인지
let 타이머ID;          // 타이머 ID

3-2. 원 만들기 함수 — 랜덤 크기 + 랜덤 위치

script.js 이어서
function 원만들기() {

  // 기존 원 제거
  let 기존원 = document.querySelector('.target');
  if (기존원) 기존원.remove();

  // 새 원 만들기
  let 원 = document.createElement('div');
  원.classList.add('target');

  // 크기 랜덤 (30px ~ 80px)
  let 크기 = Math.random() * (80 - 30) + 30;
  원.style.width  = 크기 + 'px';
  원.style.height = 크기 + 'px';

  // 위치 랜덤
  let x = Math.random() * (600 - 크기);
  let y = Math.random() * (400 - 크기);
  원.style.left = x + 'px';
  원.style.top  = y + 'px';

  // 클릭하면 점수 +1, 딜레이 후 새 원
  원.addEventListener('click', function() {
    if (!게임중) return;
    점수++;
    document.getElementById('score').innerText = 점수;

    // 랜덤 딜레이 (200ms ~ 800ms)
    let 딜레이 = Math.random() * (800 - 200) + 200;
    setTimeout(원만들기, 딜레이);
  });

  document.getElementById('game-area').appendChild(원);
}
Math.random()이 뭔가요? 0~1 사이 소수를 뽑아요. × 50 + 30 하면 30~80 사이 숫자가 돼요.
setTimeout vs setInterval? setTimeout은 한 번만, setInterval은 계속 반복해요. 원은 클릭할 때마다 한 번씩 나오니까 setTimeout이에요.

3-3. 게임 시작 + 타이머

script.js 이어서
function startGame() {
  점수 = 0;
  남은시간 = 10;
  게임중 = true;

  document.getElementById('score').innerText = 0;
  document.getElementById('timer').innerText = 10;
  document.getElementById('result').innerText = '';
  document.getElementById('start-btn').disabled = true;

  원만들기();

  // 1초(1000ms)마다 타이머 줄이기
  타이머ID = setInterval(function() {
    남은시간--;
    document.getElementById('timer').innerText = 남은시간;
    if (남은시간 <= 0) 게임오버();
  }, 1000);
}

3-4. 게임 오버

script.js 이어서
function 게임오버() {
  게임중 = false;
  clearInterval(타이머ID);

  let 원 = document.querySelector('.target');
  if (원) 원.remove();

  document.getElementById('result').innerText =
    '게임 오버! 최종 점수: ' + 점수 + '점 🎉';
  document.getElementById('start-btn').disabled = false;
  document.getElementById('start-btn').innerText = '다시 시작';
}
JS 도전 과제
  • 시간 바꾸기 — 10초 → 15초 (어디를 바꾸면 될까?)
  • 원 크기 범위 바꾸기 — 30~80 → 20~100
  • 딜레이 없애기 — setTimeout 부분 지우고 원만들기() 바로 호출
  • 점수 20점 넘으면 "클리어!" 메시지 추가하기
✅ 완성 체크리스트
  • game 폴더 안에 파일 3개가 있나요? (index.html / style.css / script.js)
  • index.html을 브라우저로 열면 게임 화면이 보이나요?
  • 게임 시작 버튼을 누르면 원이 나타나나요?
  • 원 크기가 매번 달라지나요?
  • 클릭하면 잠깐 후 새 원이 나타나나요?
  • 10초 후 게임이 끝나고 점수가 나오나요?
  • 배경색이나 원 색을 내 취향으로 바꿨나요?

AI(LLM)와 함께 게임을 만들어봐요. 코드를 다 외울 필요 없어요 — 원하는 것을 정확하게 설명하는 능력이 핵심이에요.

[ 게임 기능 ] - 게임판(600x400px) 안에 원이 랜덤 위치에 나타남 - 원 크기는 매번 랜덤 (30px~80px) - 원 클릭하면 점수 +1, 잠깐 딜레이 후 새 원 나타남 - 딜레이도 랜덤 (200ms~800ms) - 10초 타이머, 0초 되면 게임 오버 - 최종 점수 표시, 다시 시작 버튼 [ 조건 ] - 파일 3개로 분리: index.html, style.css, script.js - index.html에서 style.css와 script.js 연결 - 변수명은 영어로, 주석은 한국어로 - 어두운 배경 (예: #1a1a2e) 각 파일 코드를 따로따로 보여줘.
코드 받은 다음: index.html, style.css, script.js 파일을 game 폴더에 만들고 각각 붙여넣으면 돼요. 세 파일이 반드시 같은 폴더 안에 있어야 연결이 돼요.
주의: LLM마다 코드가 조금씩 달라요. 안 되는 부분이 있으면 오류 메시지를 그대로 붙여넣어서 다시 물어봐요.

추가하고 싶은 기능 생각하기

"게임에 뭐가 있으면 더 재밌을 것 같아요?" 자유롭게 아이디어를 내봐요.

⭐ 쉬운 것
  • 클릭할 때 원 색 랜덤 변경
  • 점수 올라갈 때 숫자 커졌다 작아지기
  • 배경색이 점점 바뀌기
  • 원 클릭 시 별 이모지 표시
⭐⭐ 중간
  • 원 여러 개 동시에 나오기
  • 점수 높을수록 원이 작아지기
  • 원 놓치면 -1점
  • 남은 시간 바에 시각화
⭐⭐⭐ 어려운 것
  • 최고 점수 기록
  • 레벨 시스템
  • 클릭 효과 애니메이션
  • 카운트다운 시작 화면

LLM에게 프롬프트 작성하기 — 이게 제일 중요해요

좋은 프롬프트의 조건 3가지:

①
내가 뭘 만들고 있는지
게임 종류, 어떻게 동작하는지
②
현재 코드 붙여넣기
AI가 코드를 봐야 정확히 알려줄 수 있어요
③
원하는 기능 구체적으로
"색 바뀌게" X → "클릭할 때 랜덤 색으로" O
❌ 나쁜 프롬프트
색 바뀌게 해줘

→ AI가 어떤 코드인지, 어디에 넣어야 하는지 몰라요.

✅ 좋은 프롬프트
나는 HTML/JS로 클릭 타겟 게임을 만들고 있어. 빨간 원이 랜덤한 위치에 나타나고, 클릭하면 점수가 올라가는 게임이야. 아래는 내 script.js 코드야: [코드 전체 붙여넣기] 여기에 "원을 클릭할 때마다 원의 색이 랜덤하게 바뀌는 기능"을 추가하고 싶어. 어느 줄에 어떤 코드를 추가하면 되는지 알려줘.
팁: 기능을 하나씩 추가해요. 여러 개 한꺼번에 요청하면 어디가 틀렸는지 찾기 어려워요.

중요: 오류 메시지를 정확하게 복사해서 붙여넣어야 해요. "오류 났어요"만 쓰면 AI가 원인을 찾지 못해요.
