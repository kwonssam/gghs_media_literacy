🗞️ 진짜일까, 가짜일까
AI로 만드는 뉴스 판별 게임

실제 있었던 AI 합성 사건에서 패턴을 스스로 찾아보고,
내가 찾은 가짜뉴스 사례를 분석한 다음, 그 패턴으로 나만의 헤드라인과 이미지를 만들어 판별 게임을 완성합니다.

🎯 이 장에서 배우는 것

학습 목표
  • 실제 사례를 분석해서 가짜뉴스 판별 패턴을 스스로 찾아낼 수 있다
  • 내가 직접 찾은 가짜뉴스 사례를 근거로 패턴을 설명할 수 있다
  • Canva·AI 이미지 생성 도구로 나만의 이미지를 직접 만들 수 있다
  • 헤드라인·이미지 데이터를 JSON으로 만들고 AI 프롬프트로 게임을 완성할 수 있다

🔍 핵심 개념 — 실제 사례로 패턴 찾기

실제 사례 ⚾ 2026년 5월, "야구장 여신" AI 영상 사건

레딧에 "Not an average Korean woman"이라는 제목으로 올라온 5초짜리 영상이 하루 만에 조회수 800만을 넘겼습니다. KBO 경기 관중석에서 흰색 상의를 입은 여성이 자연스럽게 표정을 바꾸는 모습이었고, 실제 중계 화면과 구분이 안 될 정도로 정교했어요.

✏️ 먼저 스스로 찾아보기

영상을 다시 보면서 생각해보세요. 이게 AI라는 걸 무엇을 보고 알아챌 수 있을까요? (얼굴? 배경? 화면에 적힌 글자?) 내 생각을 먼저 적어봐요.

정답을 확인해볼까요? 야구팬들은 얼굴이나 화질이 아니라 화면 속 정보에서 허점을 찾아냈습니다.

영상 속 정보실제 사실
중계 자막: 한화 김서현 vs 두산 조인성 맞대결조인성은 2017년 은퇴, 김서현은 2023년 입단 — 두 선수는 함께 뛴 적이 없음
조인성이 두산 소속 타자로 표시조인성은 현역 시절 두산에서 뛴 적이 없음
응원 문구 "최강은 두산"실제 응원 문구는 "최강 두산" — AI가 어색하게 만들어낸 표현
이 사례가 알려주는 패턴: 요즘 AI 생성물은 "그림이 이상해서" 걸리지 않고 "정보가 안 맞아서" 걸립니다. 실제로 존재하는 이름·장소·기관을 가져다가 실제로는 일어날 수 없는 조합으로 엮는 방식이에요. 이미지·영상 자체보다 화면에 적힌 이름, 날짜, 기록 같은 구체적 정보를 실제와 대조하는 습관이 훨씬 중요해졌어요.
정리 🔎 가짜뉴스 패턴, 이렇게 일반화할 수 있어요

야구장 사례처럼, 가짜뉴스는 대부분 아래 패턴 중 하나(또는 조합)로 만들어집니다.

패턴야구장 사례에서는일반적인 예시
실제 요소를 시공간적으로 불가능하게 조합 은퇴 선수와 현역 선수의 가상 맞대결 "교육부, 전국 고등학생 스마트폰 전면 수거 발표" — 기관은 진짜, 발표는 지어냄
있을 법한 숫자를 비상식적으로 부풀리기 — "○○동 편의점서 하루만에 로또 3번 당첨"
디테일에서 드러나는 어색함 응원 문구 "최강은 두산"의 어색한 번역투 실제로는 쓰이지 않는 부자연스러운 표현·문장
정리 ✅ 그래서, 뉴스를 볼 때 확인할 5가지

야구장 사례에서 확인했듯, 아래 5가지를 습관적으로 확인하면 가짜뉴스를 훨씬 잘 걸러낼 수 있습니다.

확인 항목설명
출처어느 언론사·기관인지, 처음 들어보는 곳은 아닌지
작성일오래된 기사를 최신인 것처럼 재유포한 건 아닌지
제목 톤과도하게 자극적이거나 감정을 자극하는 단어가 있는지
통계·수치출처 없는 숫자, 앞뒤가 안 맞는 통계는 아닌지
이미지·영상합성·조작 흔적, 화면 속 정보가 실제 기록과 맞는지
💡 왜 직접 만들어볼까요? 가짜뉴스를 스스로 만들어보면, 어떤 "패턴"으로 만들어지는지 몸으로 이해하게 됩니다. 그러면 나중에 진짜 가짜뉴스를 마주쳤을 때 "어, 이거 그 패턴이잖아"하고 훨씬 빨리 알아챌 수 있어요. 이번 활동의 진짜 목표는 "작성법"이 아니라 "판별력"입니다.

✍️ 나만의 가짜뉴스 찾고 만들기

1) 내가 찾은 가짜뉴스 사례

최근에 봤던 가짜뉴스, 잘못된 정보, 또는 "어? 이거 좀 이상한데" 싶었던 콘텐츠 하나를 검색하거나 기억해서 가져와보세요. 뉴스 기사, SNS 게시물, 유튜브 영상 무엇이든 좋습니다.

내가 찾은 사례 (제목/출처)해당하는 패턴내가 그렇게 판단한 이유
   
잘 모르겠다면: 검색창에 "가짜뉴스", "AI 합성 논란", "딥페이크 논란" 같은 키워드로 검색해보세요. 최근 화제였던 사례가 꽤 나올 거예요.

2) 그 패턴으로 나만의 헤드라인 만들기

예) 헤드라인 "서울시, 내년부터 대중교통 요금 150원 인상" — 판별 이유: 서울시 공식 발표 자료에 근거한 실제 정책입니다.

3) 이미지 직접 만들기 — 야구장 AI 사진처럼

야구장 사례처럼, 가짜 헤드라인에 어울리는 이미지를 직접 만들어봐요. 텍스트만 있을 때보다 훨씬 설득력 있어지고, 판별하는 재미도 커져요.

개념 이미지 파일 저장 규칙

다운로드한 이미지 파일 이름을 photo1.jpg 형식으로 바꾸고, 3단계에서 만들 news-game.html과 같은 폴더에 저장하세요. (헤드라인 번호와 맞춰서 photo1.jpg, photo2.jpg 처럼 관리하면 편해요.)

⚠️ 이미지를 만들 때 꼭 지킬 것
• 실존하는 특정 인물(친구, 선생님, 연예인 등)의 얼굴을 합성하거나 그 사람처럼 보이게 만들지 않기 — 가상의 인물, 일러스트 스타일로 만드세요.
• 이 활동은 판별 연습용입니다. 만든 이미지는 수업 안에서만 사용하고 외부(SNS 등)에 올리지 않기.
• 야구장 사례가 논란이 된 것도 실제 선수 이름과 소속을 가져다 썼기 때문이에요 — 같은 방식을 실제 상황에 쓰면 안 된다는 걸 이 활동을 통해 함께 느껴봐요.

위 2) 헤드라인 만들기에서 작성한 표를 그대로 옮기면 됩니다. 표의 한 줄 = JSON의 객체 하나예요. 아래 빈칸에 내가 쓴 헤드라인 1·2·3번을 순서대로 채워보세요.

// 내가 쓴 헤드라인 1번 → 여기
[
  {
    "headline": "[표 1번 헤드라인]",
    "isFake": [표 1번이 가짜면 true, 진짜면 false],
    "reason": "[표 1번 판별 이유]",
    "image": "[표 1번 이미지 파일명, 없으면 null]"
  },
  // 내가 쓴 헤드라인 2번 → 여기 (위와 같은 형식으로)

  // 내가 쓴 헤드라인 3번 → 여기 (위와 같은 형식으로)

]
판별 이유 쓰는 팁: 1)에서 "내가 찾은 사례"를 분석할 때 썼던 것처럼, 막연하게 쓰지 말고 구체적인 사실 근거로 씁니다. (예: "확률상 불가능" ❌ 막연함 → "3번 이상 당첨 확률은 사실상 0에 가까움" ⭕ 구체적)
주의: 문자열은 반드시 큰따옴표(")로 감싸고, 마지막 항목 뒤에는 쉼표(,)를 붙이지 않습니다. true/false/null은 따옴표 없이 그대로 씁니다.

아래 템플릿의 빈칸에 방금 만든 JSON을 붙여넣으세요.

[페르소나] 너는 고등학생을 위한 웹 게임을 만드는 프론트엔드 개발자야.
[맥락] HTML, CSS, JS가 한 파일에 다 들어가는 카드 스와이프 형식 게임을 만들 거야.
[구체적 요청]
- 화면 중앙에 뉴스 헤드라인 카드가 하나씩 뜬다
- image 값이 있으면 헤드라인 위에 <img src="파일명">으로 이미지를 보여주고, null이면 텍스트만 보여줘
- "진짜" 또는 "가짜" 버튼을 누르면 정답 여부를 확인한다
- 정답이면 초록색, 오답이면 빨간색으로 카드가 반응하고 판별 이유가 아래 표시된다
- 데이터는 아래 JSON 배열을 그대로 써라:
  (여기에 내가 만든 JSON 붙여넣기)
- 마지막에 3개 중 몇 개 맞혔는지 점수를 보여줘
[출력 형식] HTML/CSS/JS를 한 파일 안에 <style>, <script>로 넣어서 완성된 코드로만 줘
