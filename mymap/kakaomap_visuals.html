프롬프트 작성 4단계
AI에게 코드를 요청할 때 이 순서를 지키면 정확한 코드를 받을 수 있어요
① 페르소나
나는 고등학생이고, 카카오맵 _______ 서비스를 만들고 있어.
→ 페르소나 = 나를 소개하는 한 줄. AI에게 "나는 초보 고등학생"이라고 알려주면, AI가 어려운 용어 없이 쉽게 설명해줘요.
↓
② 맥락 + 현재 코드
아래는 내 현재 index.html 전체야: [ 코드 전체 붙여넣기 ]
→ 지금 내 파일을 그대로 보여줘야 AI가 엉뚱한 코드를 주지 않아요!
↓
③ 구체적 지시
아래 기능을 추가해줘:
 • 검색창에 _____ 입력 후 버튼 클릭
 • 지도에 _____로 표시, 왼쪽 리스트에 _____ 나열
 • _____ 클릭 시 _____ 팝업 표시
↓
④ 출력 형식
전체 index.html을 다시 보여줘. 변수명은 영어로, 주석은 한국어로.
공통 규칙
① 내가 뭘 만드는지 → ② 지금 코드 붙여넣기 → ③ 원하는 기능 구체적으로 → ④ 형식 지정
수업 진행 흐름
STEP 2는 생략하고, STEP 1 완성 후 바로 도전 과제로 진행합니다
STEP 1 — 뼈대 + 지도 띄우기
2분할 레이아웃 + 카카오 지도 화면에 표시 (선생님과 함께)
프롬프트 필수 항목 3가지
위치 권한 페이지가 열리면 위치 권한을 허용받아 현재 위치를 중심으로 지도에 표시
로딩 오류 방지 지도가 제대로 로드된 후에 실행되도록 해줘 (로딩 오류 방지)
JS 키 입력 카카오 JavaScript 키: [내 키 직접 입력] — 빠지면 지도가 절대 안 뜨니 반드시 넣을 것!
↓
STEP 2 — 검색 + 마커 + 팝업 (생략)
이번 수업에서는 건너뜁니다 → 도전 과제에서 직접 구현해요
↓ 바로 도전!
도전 1 — 기능 탐색 + 내 지도 기획
카탈로그에서 기능 고르기 → 주제·탭·기능 설계 → 기획서 작성
↓
도전 2 — 직접 만들기 + GitHub 배포
기획서 바탕으로 프롬프트 작성 → 구현 → GitHub Pages 배포 → 링크 제출
