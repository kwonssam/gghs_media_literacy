카카오맵 API SDK — 지도 UI 통째로 설치 프롬프트 작성 연습
API 핵심 개념 요약

NASA 실습에서 배운 내용 중 카카오맵 실습에 꼭 필요한 개념만 빠르게 정리.

API란?

외부 서버의 데이터를 가져오는 통로.
규칙대로 요청하면 데이터가 배달 옴.
🛵 배달앱처럼 주방 몰라도 주문 가능

JSON이란?

API가 데이터를 보낼 때 쓰는 포장 형식.
key : value 쌍으로 구성.
📦 data.name, data.x, data.y 로 꺼냄

API 키란?

정식 사용자임을 증명하는 코드.
키 없으면 서버가 요청 거절.
🔑 개발자 사이트에서 무료 발급

NASA API vs 카카오맵 API — 방식이 다름!
NASA는 fetch()로 URL 호출 → JSON 수신 방식.
카카오맵은 SDK 스크립트 설치 → 지도 UI + 데이터 통째로 제공하는 방식.
지도 확대·축소·마커 같은 복잡한 UI는 JSON 텍스트만으론 구현 불가 → 카카오가 미리 만든 도구 상자(SDK)를 통째로 설치해서 사용.
실습 준비 — 카카오 API 키 발급
🔑
카카오 개발자 사이트에서 JavaScript 키 발급
developers.kakao.com — 도메인 등록까지 완료해야 작동함
  • ①
    developers.kakao.com 접속 → 로그인 → 애플리케이션 추가하기
  • ②
    앱 설정 → 앱 키 탭 → JavaScript 키 복사 (REST API 키 아님!)
  • ③
    앱 설정 → 플랫폼 → Web → 사이트 도메인에 https://내아이디.github.io 입력
  • ④
    제품 설정 → 카카오맵 → 지도 사용 설정 활성화 (ON)
  • ⑤
    JavaScript 키를 메모장에 복사 보관
도메인 등록 안 하면 지도가 안 뜸. GitHub Pages 주소를 반드시 등록할 것.
실습 순서 — 2단계로 완성

한 번에 다 만들지 않음. 아래 순서대로 하나씩 추가하며 완성.

1
뼈대 + 지도 띄우기
2분할 레이아웃 + 카카오 지도 화면에 표시

왼쪽 검색창·결과 리스트, 오른쪽 지도 — 2분할 화면 구조 만들기.
카카오 지도가 학교 위치를 중심으로 뜨면 1단계 완성.

✅ 완성 기준

  • 2분할 레이아웃 (왼쪽 패널 + 오른쪽 지도)
  • 카카오 지도 화면에 표시됨
  • 지도 중심이 학교 위치로 설정됨

핵심 코드

<script src="//dapi.kakao.com/v2/maps/sdk.js?appkey=내_JS_키&libraries=services"></script>

var map = new kakao.maps.Map(el, {
  center: new kakao.maps.LatLng(37.66, 126.83),  // 학교 위도/경도
  level: 5
});
2
검색 + 마커 + 리스트 + 팝업
키워드 입력 → 마커 표시 + 결과 리스트 + 클릭 팝업까지 한 번에

검색창에 키워드 입력 → 카카오 서버에 요청 → JSON 데이터 수신 → 마커·리스트·팝업을 한 번에 표시.

전체 동작 흐름
검색 버튼 클릭 → JS가 카카오 서버에 "이 키워드로 주변 장소 알려줘" 요청 → 카카오 서버가 JSON으로 장소 목록 배달 → JS가 데이터 받아서 ① 지도에 마커 ② 왼쪽 리스트 ③ 클릭 팝업 동시에 표시.

📦 카카오가 보내주는 JSON 데이터 구조

// data[0] — 첫 번째 검색 결과 예시
{
  "place_name": "스타벅스 경기고양삼송점",     // 장소 이름
  "road_address_name": "경기 고양시 덕양구...",  // 도로명 주소
  "x": "126.8345",                          // 경도 (지도 위치)
  "y": "37.6612",                          // 위도 (지도 위치)
  "phone": "031-000-0000",                  // 전화번호
  "place_url": "http://place.map.kakao.com/..."  // 카카오맵 링크
}

핵심 코드 — 검색 + 마커 + 리스트 + 팝업

var ps = new kakao.maps.services.Places();
var infowindow = new kakao.maps.InfoWindow({ zIndex: 1 });

ps.keywordSearch(keyword, function(data, status) {
  if (status === kakao.maps.services.Status.OK) {

    data.forEach(function(place) {
      // ① 마커 찍기
      var pos = new kakao.maps.LatLng(place.y, place.x);
      var marker = new kakao.maps.Marker({ map: map, position: pos });

      // ② 리스트에 이름 추가
      var item = document.createElement('div');
      item.textContent = place.place_name;
      document.getElementById('result-list').appendChild(item);

      // ③ 마커 클릭 시 팝업
      kakao.maps.event.addListener(marker, 'click', function() {
        infowindow.setContent('<div style="padding:8px"><b>' + place.place_name + '</b><br>' + place.road_address_name + '</div>');
        infowindow.open(map, marker);
      });
    });
  }
});

✅ 완성 기준

  • 키워드 검색 시 지도에 마커 표시됨
  • 왼쪽 리스트에 장소 이름 나열됨
  • 마커 클릭 시 장소 이름·주소 팝업 표시
오류 발생 시: F12 → Console 탭 → 빨간 오류 메시지 복사 → AI에게 현재 코드와 함께 붙여넣고 수정 요청.
🏆 도전 — 탭 분리로 나만의 지도 완성

2단계까지 완성됐으면 카테고리 코드를 활용해서 탭을 나눠봐요. 키워드 검색보다 카테고리 코드로 검색하면 결과가 더 정확해요.

🔍
카테고리 코드 직접 찾기
LLM에게 물어보거나 카카오 개발자 문서에서 직접 탐색

카카오맵 API는 카테고리 코드로 장소를 검색할 수 있어요. 내가 만들고 싶은 지도에 맞는 코드를 직접 찾아봐요.

LLM 창 열고 아래처럼 질문해봐요.

전부 알려줘.
방법 2 — 카카오 개발자 문서

developers.kakao.com → 문서 → 지도/로컬 → 로컬 → 카테고리로 장소 검색

로그인 후 접속해야 상세 문서 확인 가능.

카테고리 코드 예시 (더 많은 코드는 직접 찾아볼 것!)

코드 카테고리 활용 예시

탭 분리 프롬프트 작성

카테고리 코드를 찾았으면 아래 프롬프트 템플릿에 넣어서 AI에게 요청해봐요.
반드시 현재 코드를 같이 붙여넣을 것.

① 페르소나 나는 카카오맵 장소 검색 서비스를 만들고 있는 고등학생이야. ② 맥락 + 현재 코드 아래는 내 현재 index.html 전체야: [index.html 전체 붙여넣기] ③ 구체적 지시 탭을 두 개로 나눠서 각각 다른 카테고리를 검색하게 만들어줘. 탭 1 (맛집): 카테고리 코드 FD6 (음식점)으로 지도 중심 기준 검색 탭 2 (병원): 카테고리 코드 HP8 (병원)으로 지도 중심 기준 검색 탭 전환 시 기존 마커는 지우고 새 결과만 표시해줘. ④ 출력 형식 수정·추가된 부분만 알려줘. 어느 위치에 넣으면 되는지도 설명해줘.
탭 이름이나 카테고리 코드는 자유롭게 바꿔도 됨. 맛집·병원 대신 카페·학교·편의점 등 원하는 조합으로 만들어봐요.
단계별 프롬프트 예시

① 내가 뭘 만드는지 ② 현재 코드 상태 ③ 원하는 기능 — 3가지 조건 필수.

STEP 1 — 뼈대 및 기본 지도 프롬프트

① 페르소나 나는 고등학생이고 HTML/CSS/JS를 배우고 있어. ② 맥락 카카오맵 API로 장소 검색 서비스를 새로 만들고 싶어. ③ 구체적 지시 - 왼쪽: 검색창 + 결과 리스트 / 오른쪽: 지도 — 2분할 화면 - 화면이 켜지면 우리 학교 위치(위도 37.66, 경도 126.83)를 중심으로 지도 표시 - JavaScript 키: [내 키 입력] ④ 출력 형식 index.html, style.css, script.js 3개 파일로 나눠서 보여줘. 변수명은 영어로, 주석은 한국어로.

STEP 2~3 — 기능 추가 프롬프트

① 페르소나 나는 카카오맵 장소 검색 서비스를 만들고 있는 고등학생이야. ② 맥락 + 현재 코드 아래는 내 현재 script.js야: [script.js 전체 붙여넣기] ③ 구체적 지시 검색창에 키워드 입력 후 버튼 클릭 → 카카오 장소 검색 API 요청 → 지도에 마커 + 왼쪽 리스트에 이름 표시 ④ 출력 형식 수정·추가된 script.js 부분만 알려줘. 어느 위치에 넣으면 되는지도 설명해줘.
오류 발생 시 디버깅 프롬프트:
"내 현재 코드: [붙여넣기] / 오류 메시지: [F12 Console 빨간 메시지 복사] / 원인과 수정 방법을 단계별로 알려줘."
