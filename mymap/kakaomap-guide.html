🗺️ 카카오맵 API로 행복지도 만들기

행복지도 데이터를 지도 위에 시각화하고 나만의 버전으로 업그레이드하기

카카오맵 API 지오코딩 fetch() — CSV 불러오기 프롬프트 작성 연습
오늘의 목표

전국 228개 구군의 삶의만족도 데이터를 카카오맵 위에 시각화. 마커 색깔로 행복 수준을 표현하고, 클릭하면 점수와 순위가 팝업으로 표시되는 페이지 제작.

예시 페이지 확인: 오늘 만들 결과물을 먼저 확인.
선생님이 공유한 예시 페이지 링크 접속
실습 순서
1
API 키 확인
카카오 개발자 사이트에서 두 가지 키 확인

오늘은 카카오 API 키가 두 가지 필요함. 카카오 개발자 사이트 → 내 애플리케이션 → 앱 키에서 확인.

키 종류용도사용 단계
REST API 키 주소 → 좌표 변환 (지오코딩) STEP 2 — 구글 코랩
JavaScript 키 카카오맵 지도 표시 STEP 4 — HTML 파일
두 키 모두 메모장에 복사 보관. 헷갈리지 않게 이름 적어두기.
2
지오코딩 — 주소를 좌표로 변환
구글 코랩에서 코드 실행 → 좌표 포함된 CSV 저장

지도에 핀을 찍으려면 각 구군의 위도·경도(좌표)가 필요함. 행복지도 데이터에는 구군 이름만 있으므로, 카카오 API로 주소를 좌표로 변환하는 작업 필요. 이 과정을 지오코딩이라고 함.

  • ①
    구글 코랩 접속 colab.research.google.com
  • ②
    선생님이 준 지오코딩 코드 붙여넣기
  • ③
    코드 안에서 KAKAO_API_KEY 자리에 REST API 키 입력
  • ④
    왼쪽 파일 아이콘 → 대한민국행복지도_삶의만족도.xlsx 업로드
  • ⑤
    실행 (약 2~3분 소요)
  • ⑥
    완료 후 행복지도_좌표포함.csv 다운로드
CSV 파일 구조: No, 시도, 구군, 삶의만족도, 전체주소, 위도, 경도
3
프롬프트 작성
4단계 공식으로 예시 페이지와 동일한 기능 요청

아래 예시를 참고해 직접 프롬프트 작성. ③번에 원하는 기능을 모두 포함할 것.

① 페르소나 나는 고등학생이고 HTML/CSS/JS를 배우고 있어. ② 맥락 카카오맵 API와 행복지도 CSV 데이터로 웹페이지를 새로 만들고 싶어. CSV 파일 구조: No, 시도, 구군, 삶의만족도, 전체주소, 위도, 경도 CSV 파일명: 행복지도_좌표포함.csv ③ 구체적 지시 - fetch()로 CSV 파일을 불러와서 지도에 표시해줘 - 각 구군 위치에 색깔 마커 표시 (삶의만족도 0.8 이상: 파란색, 0.6~0.8: 초록, 0.4~0.6: 노랑, 0.2~0.4: 주황, 0.2 미만: 빨강) - 마커 클릭하면 구군명, 점수, 전국 순위 팝업 표시 - 오른쪽에 구군 목록 패널 (점수 높은 순 정렬) JavaScript 키: [내 JavaScript 키 입력] ④ 출력 형식 index.html 파일 하나로 만들어줘. 변수명은 영어로, 주석은 한국어로.
JavaScript 키 자리에 반드시 내 키를 입력할 것. REST API 키가 아닌 JavaScript 키임.
4
파일 제작 및 실행
index.html과 CSV 파일을 같은 폴더에 저장
  • ①
    바탕화면에 happymap 폴더 생성
  • ②
    AI가 준 코드를 index.html로 저장해서 폴더 안에 넣기
  • ③
    아까 다운받은 행복지도_좌표포함.csv도 같은 폴더에 넣기
  • ④
    index.html 더블클릭 → 브라우저에서 실행
📁 happymap/ ├── index.html ← AI가 준 코드 붙여넣기 └── 행복지도_좌표포함.csv ← 코랩에서 다운받은 파일
CSV 파일이 반드시 같은 폴더 안에 있어야 함. fetch()로 파일을 불러오기 때문에 위치가 다르면 데이터가 표시되지 않음.
오류 발생 시: F12 → Console 탭 → 빨간 오류 메시지 복사 → AI에게 붙여넣고 수정 요청.
5
나만의 버전으로 업그레이드
기능 목록에서 선택 후 프롬프트로 추가 요청

기본 페이지 완성 후 아래 기능 중 원하는 것 선택해서 추가. 반드시 현재 코드를 함께 붙여넣을 것.

🗺️ 지도 기능 — 기본
  • 시도별 필터 드롭다운
  • 구군 이름 검색창
  • 색깔 범례 표시
  • 최고/최저 구군 자동 강조
🗺️ 지도 기능 — 도전
  • 점수별 색깔 원 오버레이
  • 마커 클릭 시 해당 구군 확대
  • 시도별 평균 점수 표시
  • 전국 순위 TOP 10 목록
📊 패널 기능 — 기본
  • 전체 평균 점수 표시
  • 구군 수 표시
  • 목록 클릭 시 지도 이동
  • 점수 높은/낮은 순 정렬 전환
📊 패널 기능 — 도전
  • 시도별 평균 막대 차트
  • 점수 분포 히스토그램
  • 다크모드 / 라이트모드 전환
  • 내 동네 검색해서 바로 이동
① 페르소나 나는 HTML/JS로 카카오맵 행복지도 페이지를 만들고 있는 고등학생이야. ② 맥락 + 현재 코드 아래는 내가 만든 index.html 전체야: [index.html 전체 붙여넣기] ③ 구체적 지시 "시도별 필터 드롭다운"을 오른쪽 패널 상단에 추가하고 싶어. 선택하면 해당 시도 구군만 목록에 표시되게 해줘. ④ 출력 형식 전체 코드 말고, 추가하거나 바꿔야 하는 부분만 알려줘. 어느 위치에 넣으면 되는지도 알려줘.
GitHub Pages에 올리기

완성한 페이지를 인터넷에 공개하려면 CSV 파일도 함께 올려야 함.

  • ①
    GitHub 저장소에 index.html과 행복지도_좌표포함.csv 둘 다 업로드
  • ②
    Settings → Pages → Branch: main / Folder: / (root) 설정
  • ③
    1~2분 후 배포 주소 확인
CSV 파일을 빠뜨리면 지도에 데이터가 표시되지 않음. index.html과 CSV 파일이 같은 위치에 있어야 함.
