위 코드를 복사해서 VS Code, 구글 Colab, 또는 파이썬 IDLE에 붙여넣고 실행하면 my_abalone.csv 파일이 생성됩니다.

  
코드 한 줄씩 이해하기

  

    

      
import pandas as pd, numpy as np

      
pandas는 표 형태 데이터를 다루는 라이브러리, numpy는 수학 계산과 난수 생성에 사용해요.

    

    

      
np.random.seed(42)

      
시드를 고정하면 코드를 다시 실행해도 같은 결과가 나와요. 친구와 결과를 비교할 때 유용합니다.

    

    

      
np.random.uniform(min, max, n)

      
min~max 사이에서 n개의 숫자를 무작위로 뽑아요. 전복의 길이나 무게처럼 연속적인 값을 만들 때 써요.

    

    

      
df.to_csv('파일명.csv', index=False)

      
DataFrame을 CSV 파일로 저장해요. index=False를 쓰면 행 번호(0,1,2...)가 파일에 포함되지 않아요.
