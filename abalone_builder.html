코드 한 줄씩 이해하기

    

      
import pandas as pd, numpy as np
pandas는 표 형태 데이터 라이브러리, numpy는 수학 계산과 난수 생성에 사용해요.

      
np.random.seed(42)
시드를 고정하면 코드를 다시 실행해도 같은 결과가 나와요. 숫자를 바꾸면 완전히 다른 데이터가 나와요.

      
np.random.uniform(min, max, n)
min~max 사이 소수를 n개 무작위로 뽑아요. 길이, 무게처럼 연속적인 값에 사용해요.

      
np.random.randint(min, max, n)
min~max-1 사이 정수를 n개 뽑아요. 나이테처럼 자연수로 세는 값에 사용해요. max는 포함 안 돼요!

      
df.to_csv('파일명.csv', index=False)
DataFrame을 CSV로 저장해요. index=False를 쓰면 행 번호가 파일에 포함되지 않아요.
